# bikikini.shop
an online shop

user can list see list of product and buy and pay with the stipe payment network
** go to Dockerfile
** run "sudo docker-compose build"
** run "sudo dcoker-compose up"
Project description
  tech stack
    - docker 
    - django
    - celery
    - redis
